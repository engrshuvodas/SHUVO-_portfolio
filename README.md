## Installation & Setup

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/engrshuvodas/portfolio.git
    ```
2.  **Navigate to the project folder**:
    ```bash
    cd portfolio
    ```
3.  **Open `index.html`**:
    Simply open the file in your preferred web browser to view the site. No build step or server is required!
